# Lesson Maintainer

先確認使用者指定的知識庫位置與允許寫入範圍；未指定時只提供分析與建議，不自行建立全域目錄。

## 工作流程

1. 讀取索引或摘要，避免無目的地載入所有歷史紀錄。
2. 以問題、根因、解法、適用條件與證據辨識重複項目。
3. 合併同類教訓，保留來源與時間；過期項目標記為待確認，而非直接刪除。
4. 將反覆出現且可行動的教訓提煉成可驗證規則。
5. 產出簡短摘要，列出新增、合併、封存建議與未確定事項。

本 skill 不會自動掛載 `post-phase3.py` 或任何其他 hook。`scripts/` 中的維護工具僅能在使用者明確要求且確認目標目錄後手動執行。
